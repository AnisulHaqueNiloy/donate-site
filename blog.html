Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each element (like headings, paragraphs, and images) is a node in that tree. This allows developers to manipulate the content, structure, and style of a webpage using languages like JavaScript

How do you select an elemant from the DOM? 

There are multiple ways to find an element from the dom such as - 

                *By ID: Select an element using its unique ID.

                *By Class: Select elements by their class name.

                *By Tag Name: Select elements by their tag name (like "p" or "div").

                *Using CSS Selectors: Use a CSS selector to find elements, either the first match or all matches.

            

          

          

            
            

                what is event delegation in the context of the DOM, and why is it useful?
            

            

              
Event delegation is a technique in the context of the Document Object Model (DOM) that allows you to manage events more efficiently by leveraging the event bubbling phase. Instead of attaching event listeners to individual elements, you attach a single event listener to a common ancestor of those elements.This ancestor then listens for events that bubble up from its children.

            

          
            

                
                

                    how do you manipulate an element's attributes and styles using the DOM?
                

                

                  
You can manipulate an element's attributes and styles in the DOM using JavaScript in a few key 


                    Attributes - 

                    *Get an Attribute: Retrieve the value of a specific attribute.

                    *Set an Attribute: Change or add a new value to an attribute.

                    *Remove an Attribute: Delete an attribute from the element.

                    *Direct Access: For common attributes (like ID or class), you can directly modify them.


                    Styles-

                    *Change a Style: Adjust the appearance of an element (like color or size).

                    *Set Multiple Styles: Apply several style changes at once.

                    *Manipulate Classes: Add, remove, or toggle CSS classes to change styles dynamically.

                    These techniques allow you to dynamically alter how elements appear and behave on a web page.
